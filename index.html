Ingestão multi‑origem

API, CSV, XLSX, ERP e MicroSaaS no mesmo padrão

API

Extração via conectores REST, com paginação, token, watermark, hash e ingestão incremental.

Arquivo

Upload manual ou automático de CSV/XLSX/JSON, com validação de layout e versão da carga.

Scraping

Selenium/Playwright apenas como extrator quando não existe API. O resultado vira arquivo ou payload padronizado.

Contexto da documentação

Este material apresenta um padrão corporativo genérico para plataforma de dados, sem nomes reais de empresa, projetos, sistemas internos ou dados sensíveis.

O objetivo é demonstrar como aplicar Data Mesh, Medallion Architecture, Data Marts, orquestração, CI/CD e camada de publicação para consumo analítico.
